# Análisis de Mortalidad SINADEF 2025

## Descripción del Proyecto
Este proyecto presenta el análisis estadístico de los registros de mortalidad del sistema SINADEF 2025, aplicando métodos de estadística descriptiva e inferencial para evaluar diferencias significativas en la distribución de las defunciones según sexo.

## Objetivo General
Aplicar pruebas de bondad de ajuste y análisis inferencial a los datos de mortalidad del sistema SINADEF 2025.

## Objetivos Específicos
- Analizar la distribución de las defunciones según sexo.
- Aplicar la prueba Chi-cuadrado.
- Comparar frecuencias observadas y esperadas.
- Interpretar los resultados desde un enfoque epidemiológico.
## Base de Datos
La información utilizada proviene del Sistema Informático Nacional de Defunciones (SINADEF) correspondiente al año 2025.

## Metodología
Se aplicó la prueba estadística Chi-cuadrado con un nivel de significancia de 0.05 para determinar si existen diferencias estadísticamente significativas en la distribución de mortalidad según sexo.

## Resultados
- Valor Chi-cuadrado: 681.74
- p-value: 2.7909E-150

Los resultados evidencian diferencias significativas en la distribución de las defunciones según sexo.

## Conclusión
La prueba de bondad de ajuste permitió demostrar que las diferencias observadas no se deben al azar, confirmando la utilidad de las pruebas inferenciales en el análisis epidemiológico y la salud pública.
